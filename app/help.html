Tài liệu hướng dẫn sử dụng Data Cleaning Tools

- Sau khi cài đặt xong, trong lần đầu tiên sử dụng, cần phải thiết lập Database bằng cách vào Main Menu > Setup Database.

Bước 1 - Khai báo File Zip chứa dữ liệu cần lọc (input).

Bước 2 - Chọn nơi lưu kết quả (output).

Bước 3 - Khai báo Batch là ký hiệu để phân biệt giữa đợt import này với đợt import trước, ví dụ: Week 1, Week 2, W1, W2, ...

Bước 4 - Nhấn nút Check Data để kiểm tra sơ bộ Data, kết quả sẽ nằm trong File Excel, gồm các thông tin như các bệnh viện không tìm thấy trong Database, các File Excel quá lớn mà hệ thống không đọc được, các File Excel bị lỗi.

Fix các lỗi nếu có ở bước 4 rồi mới tiến hành Process Data. Nếu có tên bệnh viện mới, link nó với ID của bệnh viện tương ứng trong File Excel kết quả rồi tiến hành import matches vào hệ thống bằng cách truy cập menu Main Menu > Import Matches

Nếu cần xóa Data của đợt Import khai báo ở bước 3 thì nhấn nút Clear Batch Data

Bước 5 - Import Data

Bước 6 - Generate Report
